# Udemy - Course NodeJS

### This repository is about:
1. A online Shop
You can:
* View the catalog
* View the product detail 
* Create a product
* Update a product
* Delete a product
* Add a product in the Cart
* Delete a product in the Cart 
* View the orders

The data store is in No SQL Data Base (MongoDB)

### Install
`npm install`

### Execute the Proyect
* Open a terminal: `npm start`
